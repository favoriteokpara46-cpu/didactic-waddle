## 🚀 Quick Start

### Option 1: View Locally
```bash
# 1. Clone this repository
git clone https://github.com/favoriteokpara46-cpu/didactic-waddle.git

# 2. Navigate to the folder
cd didactic-waddle

# 3. Open index.html in your browser
open index.html  # macOS
# or
start index.html # Windows
```

### Option 2: Host on GitHub Pages (Live Online) ⭐
**THIS IS THE BEST OPTION!**

1. Go to your repository settings: https://github.com/favoriteokpara46-cpu/didactic-waddle/settings
2. Scroll to **"Pages"** section (left sidebar)
3. Select **"main"** branch as source
4. Click **Save**
5. Your blog will be live at: `https://favoriteokpara46-cpu.github.io/didactic-waddle/`

✅ It usually takes 1-2 minutes to go live. Refresh the page if it doesn't appear immediately!

## 🔧 Troubleshooting

### Blog Not Showing Up?
- ⏳ Wait 2-3 minutes for GitHub to deploy
- 🔄 Refresh the page (Ctrl+Shift+Del for cache clear)
- ✅ Check that you selected "main" branch in Settings > Pages
- 🔗 Verify the URL is: `https://yourusername.github.io/didactic-waddle/`

### URL Looks Wrong?
- 👤 Make sure you're using your correct GitHub username
- 📁 Check that the repository name is exactly "didactic-waddle"
- 📄 Ensure all files (index.html, style.css, script.js) are in the main folder

### Still Having Issues?
- 📚 Check [GitHub Pages docs](https://docs.github.com/en/pages)
- 🔍 Look for error messages in Settings > Pages
- 💬 Visit [GitHub Discussions](https://github.com/favoriteokpara46-cpu/didactic-waddle/discussions)
