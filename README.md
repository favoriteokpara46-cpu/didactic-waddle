# 🎬 DramaMadeMeCryToday - Kdrama Meme Blog

Welcome to **DramaMadeMeCryToday**, your ultimate destination for Kdrama memes! This is a fun, interactive blog dedicated to celebrating the beautiful chaos and hilarious tropes of Korean dramas.

## 📋 What's Included

This blog comes with:
- ✅ **index.html** - Main blog page with posts and sections
- ✅ **style.css** - Beautiful, responsive styling
- ✅ **script.js** - Interactive features like smooth scrolling

## 🎨 Features

- 📱 **Fully Responsive** - Looks great on phones, tablets, and desktops
- 🎭 **Kdrama Meme Posts** - Pre-loaded with funny Kdrama meme posts
- 🔗 **Navigation** - Easy navigation between Home, Memes, About, and Contact sections
- ✨ **Smooth Animations** - Posts fade in beautifully when the page loads
- 🌈 **Beautiful Design** - Purple gradient header and modern styling

## 🚀 How to Use

### Option 1: View Locally
1. Download or clone this repository
2. Open `index.html` in your web browser
3. That's it! Your blog is ready to view

### Option 2: Host on GitHub Pages
1. Go to your repository settings
2. Scroll to "GitHub Pages" section
3. Select "main" branch as source
4. Your blog will be live at: `https://favoriteokpara46-cpu.github.io/didactic-waddle/`

## 📝 How to Add Your Own Memes

To add new meme posts, edit the `index.html` file and add new post cards like this:

```html
<article class="post">
    <div class="post-header">
        <h3>Your Meme Title Here</h3>
        <p class="post-date">Date Here</p>
    </div>
    <div class="post-content">
        <p>Your meme content here</p>
        <p>Add more paragraphs as needed</p>
    </div>
    <a href="#" class="read-more">Read More →</a>
</article>
```

## 🔧 Customize Your Blog

### Change Colors
Edit `style.css` and find the hex colors like `#667eea` (purple) and `#764ba2` (darker purple) and replace them with your favorite colors.

### Update Social Links
In `index.html`, find the social links section and replace `#` with your actual TikTok and Facebook links:

```html
<a href="https://www.tiktok.com/@yourprofile" class="social-btn">📱 TikTok</a>
<a href="https://www.facebook.com/yourpage" class="social-btn">👍 Facebook</a>
```
